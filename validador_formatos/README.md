# Validador de Formatos

Este proyecto contiene funciones para validar diferentes formatos de string comunes (email, DPI, tarjeta de crédito, matrícula de coche) y pruebas unitarias para cada función.

## Estructura
- `validators.py`: Funciones de validación.
- `test_validators.py`: Pruebas unitarias con pytest.

## Cómo ejecutar las pruebas

1. Instala pytest si no lo tienes:
   
   ```bash
   pip install pytest
   ```

2. Ejecuta las pruebas:
   
   ```bash
   pytest
   ```
